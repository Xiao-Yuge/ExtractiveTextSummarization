# 抽取式摘要demo
## 基本思路
1. 训练word2vec或者fasttext词向量

2. 根据训练数据计算词频

3. 方案一：对输入文本分句，使用sift的句向量生成方法计算句向量，同时计算文本向量，通过筛选与文本向量最近似的句子作为摘要句；

   方案二：参考pagerank方法引入textrank，构建句子关系图并计算句子重要性；
   
4. 将句子根据重要性与原始数据重新拼接生成抽取式摘要。
